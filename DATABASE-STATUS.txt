🚀 خطوات التشغيل في VS Code:
═══════════════════════════════════════════════════════════════

1️⃣  افتح المشروع في VS Code
    File → Open Folder → اختر مجلد المشروع

2️⃣  افتح Terminal
    View → Terminal (أو Ctrl + `)

3️⃣  تثبيت الحزم
    npm install

4️⃣  إلغاء المتغير القديم (مهم!)
    Linux/Mac:        unset DATABASE_URL
    Windows PowerShell: $env:DATABASE_URL = ""

5️⃣  مزامنة قاعدة البيانات
    npm run db:push

6️⃣  تشغيل المشروع
    npm run dev

7️⃣  افتح المتصفح
    http://localhost:3000
